# Images liste de mariage

Dépose ici une image par cadeau au format `.jpg`.

Le site convertit automatiquement le titre du cadeau en nom de fichier :
- minuscules
- sans accents
- espaces et caractères spéciaux remplacés par `-`

Exemples :
- `Siège auto Cleon` -> `siege-auto-cleon.jpg`
- `Déco pour le salon` -> `deco-pour-le-salon.jpg`
- `Commande en avant-première de GAV` -> `commande-en-avant-premiere-de-gav.jpg`
